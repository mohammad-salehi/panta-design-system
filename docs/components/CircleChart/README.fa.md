# نمودار دایره‌ای (CircleChart)

کامپوننت نمودار دایره‌ای (دونات) برای نمایش نسبت‌های یک مجموعه داده. با استفاده از کتابخانه `recharts` ساخته شده و دارای قابلیت نمایش درصدها، مجموع کل در مرکز، و راهنما (legend) است. رنگ‌ها به صورت خودکار یا سفارشی قابل تنظیم می‌باشند. با تم دارک/لایت هماهنگ می‌شود.

## Props

| نام | نوع | پیش‌فرض | توضیحات |
|------|------|---------|----------|
| `data` | `CircleChartItem[]` | **اجباری** | آرایه داده‌های نمودار (هر آیتم شامل `label`, `value`, `color?`) |
| `unit` | `string` | - | واحد مقادیر (مثلاً `'تومان'`, `'عدد'`) که در tooltip و مرکز نمودار نمایش داده می‌شود |
| `height` | `number` | `340` | ارتفاع نمودار (پیکسل) |
| `dir` | `'rtl' \| 'ltr'` | `'ltr'` | جهت متن و چیدمان راهنما |
| `className` | `string` | `''` | کلاس‌های اضافی Tailwind برای کانتینر بیرونی |

### نوع `CircleChartItem`

| فیلد | نوع | توضیحات |
|-------|------|----------|
| `label` | `string` | برچسب هر بخش |
| `value` | `number` | مقدار عددی آن بخش |
| `color` | `string` (اختیاری) | رنگ دلخواه برای آن بخش (در صورت عدم ارائه، رنگ از پالت پیش‌فرض انتخاب می‌شود) |

## نحوه استفاده

### ۱. استفاده پایه

```tsx
import { CircleChart } from 'panta_design_system';

const data = [
  { label: 'دارایی', value: 250 },
  { label: 'بدهی', value: 120 },
  { label: 'سرمایه', value: 180 },
];

<CircleChart data={data} />
```
### ۲. تنظیم واحد و جهت راست‌چین

```tsx
<CircleChart
  data={data}
  unit="میلیون تومان"
  dir="rtl"
/>
```
### ۳. ارتفاع سفارشی

```tsx
<CircleChart
  data={data}
  height={400}
/>
```
### ۴. اعمال رنگ سفارشی برای بخش‌ها

```tsx
const dataWithColors = [
  { label: 'فروش', value: 100, color: '#10b981' },
  { label: 'بازگشت', value: 30, color: '#ef4444' },
];

<CircleChart data={dataWithColors} />
```
