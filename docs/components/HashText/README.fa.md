# متن کوتاه شده (HashText)

کامپوننت نمایش متن به صورت خلاصه شده (مانند هش تراکنش‌ها، آدرس کیف پول، توکن‌ها و ...) همراه با قابلیت کپی. قسمت ابتدا و انتهای متن را نشان داده و وسط را با جداکننده (مثل سه نقطه) جایگزین می‌کند.

## Props

| نام | نوع | پیش‌فرض | توضیحات |
|------|------|---------|----------|
| `text` | `string` | **اجباری** | متن اصلی (کامل) |
| `startChars` | `number` | `6` | تعداد کاراکترهای ابتدایی که نمایش داده شود |
| `endChars` | `number` | `4` | تعداد کاراکترهای انتهایی که نمایش داده شود |
| `separator` | `string` | `'…'` | جداکننده بین ابتدا و انتهای متن |
| `className` | `string` | `''` | کلاس‌های اضافی Tailwind برای کانتینر اصلی |
| `showCopyButton` | `boolean` | `true` | نمایش دکمه کپی |
| `copyOnClickText` | `boolean` | `false` | آیا با کلیک روی خود متن نیز عمل کپی انجام شود؟ |
